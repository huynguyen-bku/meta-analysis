# LaTeX Draft Frame

This folder is a concise reading frame for manuscript writing.

## Files
- `main.tex`: entry point
- `sections/00_highlights.tex`: top-line findings
- `sections/01_scope_data.tex`: dataset profile
- `sections/02_chart_insights.tex`: short chart-based insights
- `sections/03_manuscript_blocks.tex`: reusable manuscript sentences
- `sections/04_next_steps.tex`: immediate next analyses

## Compile
From repo root:

```bash
cd latex
pdflatex main.tex
```
